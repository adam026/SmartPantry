ℹ️ Rólunk

A SmartPantry egy egyetemi alkalmazásfejlesztés beadandó keretében készült projekt, amelynek célja egy modern, konténerizált webalkalmazás és CI/CD folyamat bemutatása.

🎯 Miért készült?

A projekt célja egy olyan alkalmazás létrehozása volt, amely segít nyomon követni a hűtőben és tárolóban lévő alapanyagokat, csökkenteni az élelmiszerpazarlást, és támogatni a tudatos bevásárlást.

Az alkalmazás lehetővé teszi a készletek kezelését, bevásárlólista készítését és receptek megjelenítését a meglévő hozzávalók alapján.

🛠️ Mit szeretnék szemléltetni?

A beadandóval azt szeretném bemutatni, hogyan lehet egy teljes webalkalmazást fejlesztéstől egészen a telepítésig felépíteni modern technológiák segítségével.

  • Angular alapú frontend fejlesztés
  • ASP.NET backend és REST API
  • MongoDB adatbázis használata
  • Docker alapú konténerizáció
  • CI/CD pipeline és automatizált telepítés

🚀 A projekt célja

A SmartPantry nemcsak egy egyszerű alkalmazás, hanem egy példa arra, hogyan lehet egy valós problémára modern, skálázható megoldást készíteni.

A rendszer fejlesztése során fontos szempont volt az átlátható felépítés, a felhasználóbarát kezelőfelület és az, hogy az alkalmazás később könnyen továbbfejleszthető legyen mikroszervizes vagy Kubernetes alapú környezetre is.
